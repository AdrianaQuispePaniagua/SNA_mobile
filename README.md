
  # Mobile app for Aduanas Chile

  This is a code bundle for Mobile app for Aduanas Chile. The original project is available at https://www.figma.com/design/AhJETNQjtoDvhkeyPBaUzC/Mobile-app-for-Aduanas-Chile.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
  ## Iniciar la aplicacion

  En la raíz del proyecto ejecutar 
  ```bash
  node server/server.js
  ```
  En otra terminal
  ```bash
  npm run dev
  ```
